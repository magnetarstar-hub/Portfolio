# magnetarstar — Portfolio

Personal portfolio of **Samy Bacha**, Frontend Developer & AI Enthusiast.
Built with React 19, TypeScript, GSAP, and Framer Motion.

---

## Stack

| Layer     | Tech                                |
| --------- | ----------------------------------- |
| Framework | React 19 + TypeScript               |
| Build     | Vite                                |
| Animation | GSAP + ScrollTrigger, Framer Motion |
| Styling   | Tailwind CSS                        |
| Icons     | React Icons                         |
| Routing   | React Router DOM                    |

---

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

---

## Project Structure

```
src/
├── App.tsx          # Main layout & all sections
├── Components/      # Reusable UI components
├── assets/          # Static assets
├── index.css        # Global styles & Tailwind directives
└── main.tsx         # Entry point
```

## Sections

- **Hero** — Animated intro with custom glow background
- **Phrase** — Scroll-triggered statement section
- **Projects** — Selected work cards
- **Skills** — Tech stack chips
- **Contact** — Reach out section
